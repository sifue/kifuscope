# プロジェクト概要

Kifuscopeは、棋桜の画面共有を局面データへ変換し、YaneuraOuの評価結果をOBSへ表示する配信支援システムである。大学内イベントや実況解説で、視聴者が形勢を理解しやすくする用途を想定する。

## 倫理・不正利用防止

評価値・最善手・読み筋は配信者、実況者、OBSオーバーレイだけへ提供する。対局者へ見せない。棋桜の通信、内部API、ブラウザ内部状態は解析せず、許可された画面キャプチャだけを扱う。認識が不確かな場合は評価を停止し、推測値を表示しない。

## 非ゴール

- 対局中の意思決定支援
- 棋桜への通信介入
- YaneuraOuバイナリの再配布
- MVP 1・2における画面認識
